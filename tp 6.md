### III. LAN serveurzzzz

☀️ Déterminer sur quel port écoute le serveur NGINX

````
[wheelroot@web ~]$ sudo ss -lnpt | grep 80
LISTEN 0      511          0.0.0.0:80        0.0.0.0:*    users:(("nginx",pid=1573,fd=6),("nginx",pid=1572,fd=6))
LISTEN 0      511             [::]:80           [::]:*    users:(("nginx",pid=1573,fd=7),("nginx",pid=1572,fd=7))
````

☀️ Ouvrir ce port dans le firewall
````
[wheelroot@web ~]$ sudo firewall-cmd --permanent --add-port=80/tcp
success
[wheelroot@web ~]$ sudo firewall-cmd --reload
success
````
